HeilbaumPhotoswipeController

HeilbaumPhotoswipeController must be injected to component in constructor and is used to create a HeilbaumPhotoswipe instance.

HeilbaumPhotoswipe

Extends ionic's ViewController and thus provides all of it's methods.
Create new HeilbaumPhotoswipe instance and set some options: 

{{ code.heilbaumPhotoswipeController.example }}

create(items: Array<PhotoswipeItems>, options?: PhotoswipeOptions)

Creates a new PhotoSwipe instance by passing an array of PhotoswipeItems and optional PhotoswipeOptions.
This method returns an instance of HeilbaumPhotoswipe

PhotoswipeOptions

PhotoswipeOptions is an object of PhotoSwipe options documented on photoswipe.com .
Example: 

{{ code?.photoswipeOptions?.example }}

PhotoswipeItem

PhotoswipeItem declares an object type with all properties possible in PhotoSwipe.

Please consider: If src is set, w and h is required
