Getting Started

Heilbaum Ionic PhotoSwipe Component

What does it do?

This component provides the fabulous PhotoSwipe gallery within ionic 2.
For more information about PhotoSwipe see photoswipe.com.

Installation

Installation is somehow tricky, but this way of installation provides some more flexibility, as you are able to customize PhotoSwipe's modules and it's template.

  1. Install component via npm install heilbaum-ionic-photoswipe --save
  2. Add HeilbaumPhotoswipeModule to imports array in src/app/app.modules.ts' NgModules decorator.
  3. Create file config/copy.config.js and insert: 
    {{ code.copyConfigJs }}
    or ad least add content of photoSwipe to your own copy.config.js.
    If you use a customized photoswipe.js and/or photoswipe-ui-default.js, update the src-path.
  4. Add 
    {{ code.packageJson }}
    to your package.json

Usage

  1. Inject HeilbaumPhotoswipeController to your component: 
    {{ code.componentConstructor }}
  2. And add this method to your component: 
    {{ code.componentSingleThumbnail }}
    where this.photoswipeItems is an array of objects of type PhotoswipeItem
  3. Add gallery thumbnail to html: 
    {{ code.componentTemplate }}

You may find some more example of implementations on page Example and it's code on GitHub.
